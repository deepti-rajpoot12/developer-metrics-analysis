# Developer Performance Dashboard

## Overview

A full-stack dashboard that analyzes developer productivity using Excel data.

## Features

* Metrics: Cycle Time, Lead Time, Bug Rate, Deployment Frequency
* Problem detection + actionable insights
* Team filter
* Bar chart visualization

## Tech Stack

* Frontend: React
* Backend: Node.js (Express)
* Data: Excel (XLSX)

## How to Run

### Backend

cd backend
node server.js

### Frontend

cd my-app
npm run dev
