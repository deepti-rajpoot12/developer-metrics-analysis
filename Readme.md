# Developer Performance Dashboard

## Overview
Full-stack dashboard to analyze developer performance using React, Node.js, and Excel data.

## Features

* Metrics: Cycle Time, Lead Time, Bug Rate, Deployment Frequency
* Problem detection + actionable insights
* Team filter
* Bar chart visualization

## Tech Stack

* Frontend: React
* Backend: Node.js (Express)
* Data: Excel (XLSX)
*  charts

## How to Run ->

### Backend

cd backend
node server.js

### Frontend

cd my-app
npm run dev
